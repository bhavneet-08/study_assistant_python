# 📚 Study Assistant — Python Project

A command-line quiz application built in Python by a +2 student from Ludhiana, India.

## 💡 Why I Built This
As a student in India, I noticed many students don't have access to 
interactive study tools — especially without internet. I built this 
tool to help students like me practice and track their progress.

## ✨ Features
- Quiz on multiple subjects (Biology, Physics, History)
- Tracks your name, score, percentage and date
- Saves your score history permanently
- Simple menu — anyone can use it

## 🚀 How to Run
1. Make sure Python is installed on your computer
2. Download app.py
3. Open terminal and type: python app.py
4. Choose a topic and start the quiz!

## 🛠️ Built With
- Python 3
- File Handling
- Dictionaries
- Loops & Functions
